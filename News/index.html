Ubiquiti UniFi CVE-2026-22557 (CVSS 10.0) — Unauthenticated Path Traversal Enables Full Account Takeover Act Today Score: 11/11

CVE-2026-22557 is a maximum-severity (CVSS 10.0) unauthenticated path traversal flaw in UniFi Network Application versions prior to 10.1.89. An attacker with network access can traverse directory boundaries, access OS-level files, and achieve full account takeover. A companion privilege-escalation flaw (CVE-2026-22558, CVSS 7.7) can be chained with it; Ubiquiti issued an emergency patch requiring update to 10.1.89+ (official) or 10.2.97+ (RC) and UniFi Express firmware 4.0.13+.

Check your small-business client’s UCG Fiber gateway version and update to UniFi Network Application 10.1.89+ immediately — their Ubiquiti device is directly exposed to this CVSS 10.0 unauthenticated exploit.
