Client Alert Templates

Subject: Security Alert: Do Not Open Unexpected Remote Desktop (.rdp) File Attachments 
Hi team,

We want to flag an active phishing campaign that uses Remote Desktop files (.rdp) as email attachments. If you receive an email with a .rdp file attached — even if it appears to come from a trusted sender — do not open it.

Here is what to watch for:
• An unexpected email containing a .rdp file attachment
• A pop-up dialog box prompting you to connect to a remote computer you did not request
• Any request to allow remote access to your files, clipboard, or drives through a connection prompt you did not initiate

If you accidentally open a .rdp file and see a remote desktop connection dialog, click Cancel immediately and contact us.

Our team will never send you a .rdp file by email. If you receive one or are unsure about an attachment, forward it to us before opening it.

Thanks,
Larry
